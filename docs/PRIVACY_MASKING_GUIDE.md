# 개인정보 마스킹 가이드

공개 이미지에는 사용자 이름, 프로필 사진, 이메일, 조직 이름, 알림, 브라우저 북마크, 로컬 경로, 토큰과 API 키를 남기지 않습니다.

1. 데모 저장소만 사용합니다.
2. 캡처 직후 원본을 비공개 폴더에 보관합니다.
3. 공개본은 이름·아바타·URL 쿼리·알림을 가리고, 필요한 버튼만 확대합니다.
4. PR 본문과 Commit 메시지에도 실제 고객·동료 이름을 쓰지 않습니다.
5. 배포 전 이미지 URL을 열어 원본 경로가 남아 있지 않은지 확인합니다.

민감정보가 이미 Push됐다면 이미지를 지우는 것으로 끝나지 않습니다. 키를 즉시 폐기·재발급하고 관리자에게 알립니다.
