# 콘텐츠 근거와 확인일

| 주제 | 근거 | 확인일 |
| --- | --- | --- |
| Branch, Commit, Push, Pull | GitHub Desktop의 기본 작업 흐름 | 2026-07-23 |
| Pull Request, 리뷰, 병합 | GitHub 웹의 Pull requests 화면 | 2026-07-23 |
| 초대와 권한 | GitHub 저장소 Settings의 Collaborators 화면 | 2026-07-23 |

화면은 GitHub UI 변경에 따라 달라질 수 있습니다. 변경이 확인되면 화면 캡처, alt 텍스트, 본문 설명, metadata를 함께 갱신합니다.
